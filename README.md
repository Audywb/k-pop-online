# k-pop-online
โปรเจคขายสินค้า K-POP

# ติดตั้ง
1. Python 3.10
2. Mongodb compass
3. ติดตั้ง Python pip
- curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py
- python get-pip.py
4. Visual Studio Code
5. ติดตั้ง github && git clone
6. ติดตั้ง Library
- pip install -r requirements.txt
# RUN
- python app.py
